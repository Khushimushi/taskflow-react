# TaskFlow

A full-stack task management application built using React, Express and MongoDB.

## Features

- Create Tasks
- View Tasks
- Update Task Status
- Delete Tasks
- MongoDB Atlas Integration
- User Registration with Password Hashing

## Tech Stack

### Frontend
- React
- React Router
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- bcryptjs

## Project Structure

TaskFlow
├── frontend
|── backend

## Installation

### Frontend

npm install
npm run dev

### Backend

npm install
npm run dev
